== Description ==

Adds a customizable signature under posts, pages and feed content. Language: english and german.

Deutsch:
F&uuml;gt eine frei definierbare Signatur unter Beitr&auml;gen, Seiten und in den RSS Feed ein. Ideal f&uuml;r Copyrighthinweise usw. Deutsche und englische Sprachdatei vorhanden.

More informations on the official plugin page: [http://sxss.nw.am](http://sxss.nw.am "sxss.nw.am")

== Installation ==

Upload the sxss Signature plugin to your blog, activate it, go to 'Settings > sxss Signature'.
